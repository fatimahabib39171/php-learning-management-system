# PHP Learning Management System

A Learning Management System (LMS) developed using PHP and MySQL to manage courses, students, and instructors.

## Features
- User authentication
- Role-based access (Admin / Teacher / Student)
- Course management
- Student enrollment
- Upload learning materials
- Dashboard for users

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript

## How to Run
1. Install XAMPP
2. Copy project folder to `htdocs`
3. Import database in phpMyAdmin
4. Update database credentials in `config.php`
